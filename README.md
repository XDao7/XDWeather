## 说明

gradle.properties文件已被移除，请自行添加如下参数：

签名文件相关：

KEY_PATH=

KEY_PASS=

ALIAS_NAME=

ALIAS_PASS=

和风天气私钥（注：需要开发者账号私钥，个别接口普通用户账号不支持）：

PRIVATE_QWEATHER_KEY=
